# Growth Distribution Playbook (Week 1)

## UTM links

Use one canonical campaign for launch:

- `utm_source` = platform name (`x`, `reddit`, `producthunt`, `hackernews`, `facebook`, `linkedin`)
- `utm_medium` = `post`
- `utm_campaign` = `launch_week_1`

Example:

`https://aireplytool.com/?utm_source=x&utm_medium=post&utm_campaign=launch_week_1`
